# Coaching analysis rubric (4 phases × 5 lenses)

For each **finding**, ask:

1. Is it backed by a specific number or timestamp?  
2. Did it meaningfully affect the outcome?  
3. Can you give a concrete alternative action?

If any answer is no, skip it.

## Four phases

1. **Draft / Pre-game** — comp identity, win conditions, power spikes, AD/AP balance  
2. **Early (0–14 min)** — laning, first objectives, jungle pathing, first blood, CS  
3. **Mid (14–25 min)** — objective trades, roaming, teamfight initiation, towers  
4. **Late (25+ min)** — Baron/Elder, inhibitors, teamfights, win condition  

## Five lenses (each phase)

1. **Decision audits** — drake ~5, Herald ~8, second drake, Baron ~20, aces: what did each team do?  
2. **Resource efficiency** — CS/min bands (support ~1, solo ~7–9, jungle ~5–7); vision/min (support ≥2.5, jungle ≥1.0); damage per gold.  
3. **Objective priority** — traded worse objective for better?  
4. **Teamfight breakdowns** — 3+ kills in ~60s: engage quality, first death, gold swing.  
5. **Win condition adherence** — comp win con (teamfight, pick, split, scale, poke) vs actual play.  

## Tone

- Cite timestamp, number, or name; no vague claims.  
- Honest about team vs personal fault.  
- Credit good play.  
- Don’t pad; “No major issues detected in this phase” is fine.  
- Practice focuses must be specific to **this** game.
